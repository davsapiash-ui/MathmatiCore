# MathmatiCore

A pedagogical math web application designed for elementary school students (Grades 1-3).
This project provides interactive virtual manipulatives (base-10 blocks) and a number line to teach foundational math concepts like place value, flexible decomposition, regrouping, and addition.

## Features
* **Place Value Board:** Interactive columns for units, tens, hundreds, and thousands.
* **Flexible Decomposition:** Allows students to ungroup larger blocks (e.g., 100 into 10 tens) without forced auto-grouping, promoting a deep understanding of numbers.
* **Socratic Assistant:** Context-aware prompts and scaffolding questions.
* **Silent Radar:** Tracks cognitive struggles and interaction patterns silently in the background.

## Technology Stack
* HTML5
* CSS3 (Vanilla)
* JavaScript (ES6 Modules)
* Firebase Hosting
